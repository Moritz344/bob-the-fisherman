# Usage
```bash
git clone https://github.com/Moritz344/bob-the-fisherman.git
cd bob-the-fisherman/
```
There is a cli and gui.

## CLI
Located at `bob-the-fisherman/cli/` - Bun project.
```bash
cd bob-the-fisherman/
cd cli/
bun install
bun run index.ts
```
